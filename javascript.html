What is JavaScript?

JavaScript is a lightweight, interpreted programming language that runs directly in the web browser. It is the only programming language natively supported by all browsers, making it the essential tool for adding behaviour and interactivity to web pages.

While HTML defines what is on the page and CSS defines how it looks, JavaScript defines what it does. It can respond to user events (clicks, typing, scrolling), modify the page content dynamically, validate form data before submission, fetch data from servers without reloading the page (AJAX), and much more.

JavaScript was originally designed for the browser but now also runs on servers (Node.js), in mobile apps (React Native), and in desktop applications (Electron). It is one of the most widely used programming languages in the world.

Variables & Data Types

Variables are named containers that store data values. JavaScript has three ways to declare a variable, each with different scoping and mutability rules:

  • var — the original declaration keyword. Function-scoped and can be re-declared. Mostly replaced by let and const in modern JavaScript due to confusing hoisting behaviour.
  • let — block-scoped and can be reassigned. The standard choice for values that need to change.
  • const — block-scoped and cannot be reassigned after declaration. Use for values that should not change (including most function references and DOM elements).
// Primitive data types let name = "Prabin"; // String — text in quotes let age = 21; // Number — integers and decimals let isEnrolled = true; // Boolean — true or false let score = null; // Null — intentionally empty let result; // Undefined — declared but not assigned // Reference types const colours = ["red", "green", "blue"]; // Array — ordered list const student = { // Object — key-value pairs name: "Prabin", age: 21, course: "CET138" }; // Template literals — embed variables in strings console.log(`Hello, my name is ${name} and I am ${age} years old.`);

JavaScript uses dynamic typing — you do not declare a variable's type; JavaScript infers it from the value. You can check a value's type with typeof variable.

Functions

A function is a reusable, named block of code that performs a specific task. Functions are fundamental to JavaScript — they reduce repetition, make code easier to test, and allow you to break complex logic into smaller, manageable pieces.

// Function declaration — hoisted, available before its definition function greet(name) { return "Hello, " + name + "!"; } // Function expression — assigned to a variable const add = function(a, b) { return a + b; }; // Arrow function (ES6) — concise syntax, no own 'this' const multiply = (a, b) => a * b; // Default parameters — used when no argument is passed function welcome(name = "Student") { return `Welcome, ${name}!`; } // Rest parameters — accept any number of arguments function sum(...numbers) { return numbers.reduce((total, n) => total + n, 0); } console.log(greet("Prabin")); // Hello, Prabin! console.log(add(3, 4)); // 7 console.log(multiply(5, 6)); // 30 console.log(welcome()); // Welcome, Student! console.log(sum(1, 2, 3, 4)); // 10

The DOM & DOM Manipulation

JavaScript accesses and modifies the DOM — a live tree of every element on the page.

The DOM (Document Object Model) is a programming interface that represents an HTML document as a tree of objects. Every HTML element becomes a node in this tree, and JavaScript can read and modify these nodes to dynamically update what the user sees — without reloading the page.

// Selecting elements const heading = document.getElementById("title"); const buttons = document.querySelectorAll(".btn"); // Returns NodeList const firstCard = document.querySelector(".card"); // First match // Reading and changing content heading.textContent = "Updated Title"; // Change text (safe) heading.innerHTML = "<em>Italic</em> Title"; // Parses HTML (use carefully) // Changing styles heading.style.color = "blue"; heading.style.fontSize = "2rem"; // Adding and removing CSS classes firstCard.classList.add("active"); firstCard.classList.remove("hidden"); firstCard.classList.toggle("expanded"); // Add if absent, remove if present // Reading and setting attributes const link = document.querySelector("a"); link.setAttribute("href", "https://example.com"); console.log(link.getAttribute("href")); // Creating and inserting new elements const newPara = document.createElement("p"); newPara.textContent = "I was created by JavaScript!"; document.body.appendChild(newPara);

Events & Event Listeners

Events are actions that happen in the browser — a user clicking a button, typing in a field, scrolling the page, or the document finishing loading. JavaScript uses event listeners to detect these events and run a function (called an event handler) in response.

const btn = document.getElementById("submit-btn"); // addEventListener(event, handler) is the modern, preferred approach btn.addEventListener("click", function(event) { event.preventDefault(); // Stop the default action (e.g. form submit) console.log("Button clicked!"); }); // The event object carries information about what happened document.addEventListener("keydown", function(event) { console.log("Key pressed:", event.key); if (event.key === "Enter") { /* do something */ } }); // Common events: // click, dblclick — mouse clicks // keydown, keyup, keypress — keyboard // submit — form submission // input, change — form field changes // mouseover, mouseout — hover // load, DOMContentLoaded — page/resource loading // scroll, resize — viewport changes

Always use addEventListener rather than inline HTML event attributes like onclick="..." — it keeps your JavaScript separate from your HTML and allows multiple listeners on the same element.

Conditionals & Loops

Conditionals allow your code to make decisions and execute different paths based on conditions. Loops allow you to repeat code efficiently without writing it multiple times.

If / Else If / Else

let score = 72; if (score >= 70) { console.log("Pass — well done!"); } else if (score >= 50) { console.log("Borderline — keep working."); } else { console.log("Fail — please re-sit."); } // Ternary operator — shorthand for simple if/else const grade = score >= 70 ? "Pass" : "Fail";

Loops

// for loop — when you know the number of iterations for (let i = 0; i < 5; i++) { console.log("Iteration", i); } // while loop — when you loop until a condition changes let count=0; while (count < 3) { console.log("Count:", count); count++; } // for...of — loop over an array's values const fruits=["apple", "banana" , "cherry" ]; for (const fruit of fruits) { console.log(fruit); } // forEach — array method with a callback fruits.forEach(function(fruit, index) { console.log(index, fruit); });

Arrays & Objects

Arrays and objects are the two main ways to group and structure data in JavaScript. Mastering them is essential for working with any real application.

// Arrays — ordered, indexed collections const modules = ["HTML", "CSS", "Bootstrap", "JavaScript"]; modules.push("Node.js"); // Add to end modules.pop(); // Remove from end modules.unshift("Intro"); // Add to start const first = modules[0]; // Access by index // Useful array methods const upper = modules.map(m => m.toUpperCase()); // Transform each item const web = modules.filter(m => m !== "Intro"); // Keep matching items const found = modules.find(m => m === "CSS"); // Find one item // Objects — key-value stores const student = { name: "Prabin", age: 21, course: "CET138", greet() { // Method — function inside an object return `Hi, I am ${this.name}`; } }; console.log(student.name); // Dot notation console.log(student["age"]); // Bracket notation student.grade = "A"; // Add a new property // Destructuring — extract values into variables const { name, course } = student; const [first2, second] = modules;
