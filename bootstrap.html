Bootstrap Framework

What is Bootstrap?

Bootstrap is a free, open-source front-end CSS framework originally developed by Twitter in 2011. It provides a comprehensive collection of pre-built CSS classes, layout tools, and UI components that allow developers to build responsive, professional-looking websites quickly — without writing all their styles from scratch.

Bootstrap's key strength is its responsive grid system, which makes it straightforward to build layouts that automatically adapt to any screen size. Instead of writing custom media queries for every breakpoint, you simply apply Bootstrap's utility classes directly in your HTML.

Bootstrap 5 (the current major version) removed the jQuery dependency, making it lighter and faster than previous versions. It is loaded via a CDN link — no installation required:

<!-- Add to <head> for CSS --> <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css"> <!-- Add before </body> for JavaScript components (modals, dropdowns, etc.) --> <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>

The Bootstrap Grid System

Bootstrap's grid divides every row into 12 equal columns, adapting to screen width.

Bootstrap's grid is built on CSS Flexbox and divides every row into 12 columns. You control how many columns each element occupies at each screen size by combining the container, row, and column classes. This creates layouts that reflow automatically as the screen gets smaller.

Bootstrap defines five breakpoints — thresholds where the layout changes:

  • xs — Extra small: <576px (default, no suffix: col-)
  • sm — Small: ≥576px (col-sm-)
  • md — Medium: ≥768px (col-md-)
  • lg — Large: ≥992px (col-lg-)
  • xl / xxl — Extra large: ≥1200px / ≥1400px
<div class="container"> <div class="row"> <!-- Full width on mobile, half on tablet, third on desktop --> <div class="col-12 col-md-6 col-lg-4">Column 1</div> <div class="col-12 col-md-6 col-lg-4">Column 2</div> <div class="col-12 col-md-12 col-lg-4">Column 3</div> </div> </div>

The container class centres content and adds horizontal padding. Use container-fluid for a full-width container. The row class creates a flex row with negative margins to counteract column padding. Each col-* class occupies that many of the 12 available columns.

Utility Classes

Bootstrap's utility classes are single-purpose CSS classes that apply one specific style. Instead of writing custom CSS for common tasks, you compose multiple utility classes directly on your HTML elements. This approach, known as utility-first, speeds up development considerably.

Spacing — Margin & Padding

Bootstrap uses a scale of 0–5 for spacing values (based on a 0.25rem unit), with shorthand for margin (m) and padding (p). You can target all sides, top, bottom, left, right, or horizontal/vertical.

<div class="mt-3 mb-2 px-4 py-2">...</div> <!-- mt-3 = margin-top: 1rem (3 × 0.25rem × 4) --> <!-- mb-2 = margin-bottom: 0.5rem --> <!-- px-4 = padding left & right: 1.5rem --> <!-- py-2 = padding top & bottom: 0.5rem --> <div class="mx-auto">Centred horizontally</div> <div class="ms-auto">Pushed to the right</div>

Typography Utilities

<p class="text-center text-primary fw-bold">Blue bold centred text</p> <p class="text-danger fs-6">Small red text</p> <p class="text-muted text-uppercase ls-wider">Muted uppercase</p> <p class="lead">Larger introductory paragraph</p>

Display & Flexbox Utilities

<div class="d-flex justify-content-between align-items-center"> <span>Left item</span> <span>Right item</span> </div> <!-- Hide on mobile, show on desktop --> <div class="d-none d-md-block">Desktop only</div> <!-- Show on mobile, hide on desktop --> <div class="d-block d-md-none">Mobile only</div>

Colour Utilities

<p class="text-primary">Blue</p> <p class="text-success">Green</p> <p class="text-danger">Red</p> <p class="text-warning">Yellow</p> <div class="bg-light p-3">Light background</div> <div class="bg-dark text-white p-3">Dark background</div>

Bootstrap Components

Bootstrap provides a library of pre-built UI components — interactive and styled elements that would otherwise require significant CSS and JavaScript to build from scratch. You simply apply the correct Bootstrap classes to your HTML markup.

Buttons

Bootstrap button classes apply colour, padding, border-radius, and hover/focus states automatically. The btn class is always required as a base, with a context modifier like btn-primary or btn-outline-danger added alongside it.

<button class="btn btn-primary">Primary</button> <button class="btn btn-secondary">Secondary</button> <button class="btn btn-success">Success</button> <button class="btn btn-danger btn-sm">Small Danger</button> <button class="btn btn-outline-primary btn-lg">Large Outlined</button>

Alerts

Alert components are used to display contextual feedback messages. They support dismissal when combined with the Bootstrap JavaScript bundle.

<div class="alert alert-success">Form submitted successfully!</div> <div class="alert alert-danger">Please correct the errors below.</div> <div class="alert alert-warning alert-dismissible fade show"> Warning: Your session will expire soon. <button type="button" class="btn-close" data-bs-dismiss="alert"></button> </div>

Cards

Cards are flexible content containers with optional header, body, and footer sections. They are one of Bootstrap's most versatile components, used for product listings, blog previews, user profiles, and more.

<div class="card" style="width: 18rem;"> <img src="photo.jpg" class="card-img-top" alt="..." /> <div class="card-body"> <h5 class="card-title">Card Title</h5> <p class="card-text">Description of the card content.</p> <a href="#" class="btn btn-primary">Read More</a> </div> </div>

Navbar

Bootstrap's navbar component is fully responsive — it collapses into a hamburger menu on mobile screens automatically using the navbar-expand-lg class and Bootstrap's JavaScript bundle.

<nav class="navbar navbar-expand-lg navbar-light bg-light"> <div class="container"> <a class="navbar-brand" href="#">MySite</a> <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarNav"> <span class="navbar-toggler-icon"></span> </button> <div class="collapse navbar-collapse" id="navbarNav"> <ul class="navbar-nav ms-auto"> <li class="nav-item"><a class="nav-link active" href="#">Home</a></li> <li class="nav-item"><a class="nav-link" href="#">About</a></li> </ul> </div> </div> </nav>

Bootstrap Form Classes

Bootstrap provides a set of form classes that style inputs, labels, and groups consistently across all browsers. Key classes include form-control for inputs and textareas, form-label for labels, form-select for dropdowns, and mb-3 to space out form groups.

<div class="mb-3"> <label for="email" class="form-label">Email address</label> <input type="email" class="form-control" id="email" placeholder="name@example.com" /> <div class="form-text">We'll never share your email.</div> </div> <div class="mb-3"> <label for="course" class="form-label">Course</label> <select class="form-select" id="course"> <option>Choose...</option> </select> </div> <div class="form-check mb-3"> <input class="form-check-input" type="checkbox" id="agree" /> <label class="form-check-label" for="agree">I agree</label> </div>
